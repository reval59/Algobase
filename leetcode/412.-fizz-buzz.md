# 412. Fizz Buzz

## Problem

Write a program that outputs the string representation of numbers from 1 to n.

But for multiples of three it should output “Fizz” instead of the number and for the multiples of five output “Buzz”. For numbers which are multiples of both three and five output “FizzBuzz”.

**Example:**

```text
n = 15,

Return:
[
    "1",
    "2",
    "Fizz",
    "4",
    "Buzz",
    "Fizz",
    "7",
    "8",
    "Fizz",
    "Buzz",
    "11",
    "Fizz",
    "13",
    "14",
    "FizzBuzz"
]
```

## Solution

```cpp
class Solution {
public:
    vector<string> fizzBuzz(int n) {
        vector<string> ans;
        ans.reserve(n);
        for (int i = 0; i < n; i++) {
            int fizz = ((i + 1) % 3 == 0);
            int buzz = ((i + 1) % 5 == 0);
            int result = (buzz << 1) + fizz;
            string str;
            switch (result) {
                case 0:
                    str = to_string(i + 1);
                    break;
                case 1:
                    str = "Fizz";
                    break;
                case 2:
                    str = "Buzz";
                    break;
                default:
                    str = "FizzBuzz";
            }
            ans.push_back(str);
        }
        return ans;
    }
};

```
